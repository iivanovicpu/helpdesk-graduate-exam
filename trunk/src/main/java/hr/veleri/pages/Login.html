Pred Vam je aplikacija pod nazivom "Helpdesk" koja predstavlja praktični dio zadatka završnog rada. Naslov završnog rada je: "Primjena tehnologija Java 2 Enterprise Edition platforme za razvoj poslovnih aplikacija". Praktični dio zadatka ima za cilj pokazati osnovne elemente koje mora sadržavati suvremena Web bazirana poslovna aplikacija. Namjena ove aplikacije detaljno je opisana u pismenom dijelu zadatka.

Aplikacija započinje prijavom korisnika u sustav. Ovime se identificira korisnik a unutar same aplikacije omogućuje se i autorizacija prema tipu korisnika. Ugrađeno je tri tipa korisnika: ADMINISTRATOR, KLIJENT, PODRŠKA Radi uvida u funkcionalnosti aplikacije moguće je prijaviti se u sustav s nekim od slijedećih podataka:
